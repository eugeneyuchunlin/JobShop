# Developement Note

## Arguments

* crossover: means cross over rate
	* `--crossover=` 	
	* `-cr`
	* default : 0.6
* mutation : means mutation rate
	* `--mutation=`	
	* `-m`
	* default : 0.4
* chromosomes : number of chromosomes
	* `--chromosomes=`
	* `-c`
	* default : 50
* elite : elite rate
	* `--elite`
	* `-e`
	* default : 0.4
